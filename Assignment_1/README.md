This program uses an array to store information from an input file. When the program is run a menu will pop-up giving the user 5 options.
1. Check for an integer (checkes the array for the integer and returns the index or indexs of the integers or returns -1 if it's not in the array) 
2. Modify a Value at an Index (Edits the value of an integer at an index, returns the old and new values at the index)
3. Add an Integer to the End of Array (Adds an integer at the end of the array)
4. Remove a Value at an Index (Removes the value of an integer at an index)
5. End (Ends program)
